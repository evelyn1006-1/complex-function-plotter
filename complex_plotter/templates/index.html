Complex analysis workspace

Complex Function Plotter

Plot or integrate complex functions of z. Component variables x = Re(z) and y = Im(z) are available. Piecewise forms work with where(cond, a, b) and piecewise(cond1, val1, cond2, val2, default). In integration mode, click inside the plot to build a path from line, circle, arc, Bézier, polyline, or ray segments.

Calculator feel, complex-plane power. Use scroll zoom and pan in plot modes. Integration mode turns the plot into a precise click-to-place path editor.
Pan or scroll-zoom the plot, then press Plot / Compute to resample the currently visible region.
